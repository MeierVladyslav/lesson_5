• git add
  • lgit status
  • git commit
  1. Команда git add добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита.
  2. Команда git status показывает состояния файлов в рабочем каталоге и индексе: какие файлы изменены, но не добавлены в индекс; какие ожидают коммита в индексе.
  3. Команда git commit берёт все данные, добавленные в индекс с помощью git add, и сохраняет их слепок во внутренней базе данных, а затем сдвигает указатель текущей ветки на этот слепок.
